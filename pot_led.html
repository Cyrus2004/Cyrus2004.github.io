Potentiometer-controlled LED project

The LED is connected to the Arduino's pin 5 in series with a 220Ohm resistor, which again is connected to GND. A seperate trimmer gives signal to the analog input A1. Schematics and code can be found at the botton of the page.

You will need:

  • An Arduino and a compatible USB
  • A solderless breadboard
  • Some wire
  • Cutting pliers
  • 1 trimmer
  • 1 LED
  • 1 220Ohm resistor

Instructions

  1. Connect a piece of wire between pin 5 and column 9 on the breadboard.
  2. Place the LED between column 9 and 10.
  3. Place the 220Ohm resistor between column 10 and the minus-row.
  4. Add the trimmer to the circuit, with pin A connected to column 6, W connected to column 5, and pin B connected to column 4.
  5. Make a connection between column 4 and the plus-row.
  6. Make a connection between pin A1 and column 5.
  7. Connect column 6 and the minus-row together.
  8. Connect the plus-row to 5V on the Arduino, and the minus-row to GND on the Arduino.
  9. Upload and run the Arduino program!(See code below)

Code

const int pinPot = A1;

const int pinLED = 5;


void setup(){

pinMode(pinPot, INPUT);

pinMode(pinLED, OUTPUT);

}


int sensor = 0; //Stores inndata from the trimmer.

int brightness = 0; //Stores output value going to LED.


void loop(){

sensor = analogRead(pinPot);

brightness = map(sensor, 0, 1023, 0, 255); //Translates from 11-bit signal from the trimmer, to 8-bit duty-cycle going to LED.

analogWrite(pinLED, brightness);

}
