Running lights project

This project uses an Arduino to light up LEDs in a LED stripe, one after another, so you get a ''dot'' that moves across the line. The speed the dot moves, is determined by the position of a trimmer. The anode on each light is connected with wires to their own digital pin on the Arduino, with a common GND for all the cathodes on each of them. A seperate trimmer gives signal to the analog input A0. Schematics and code can be found at the botton of the page.

You will need:

  • A soldering iron and some soldering tin
  • An Arduino and a compatible USB
  • A vero board
  • Some wire
  • Cutting pliers
  • 1 trimmer
  • 7 LEDs
  • 1 220Ohm resistor

Instructions

  1. Solder the LEDs to the vero board, with a one to three hole distance. Leave a row for the wires coming from the Arduino.
  2. De-insulate seven pieces of wire, and place each of them in the holes before every LDR, before seperately soldering the LDRs and the wires together.
  3. Connect the anodes of each LED together in paralell by making a stripe of soldering tin across them.
  4. Solder a 220Ohm resistor between anywhere on the stripe of soldering tin and the next row(Please don't follow my bad example in the background image).
  5. Solder a piece of wire to the resistor. This will be the GND.
  6. Solder one piece of wire to each of the three terminals on the trimmer.
  7. Connect the wire attached to "A" to GND on the Arduino, connect "B" to 5V, and "W" to pin A0.
  8. Attach the wires soldered to each of he LDRs' anodes to pins D4 to D10
  9. Connect the wire attached to the resistor to the other GND on the Arduino.
  10. Upload and run the Arduino program!(See code below)

int pinA = 4; //Defines the lowest output pin value.

int pinB = 10; //Defines the highest output pin value.

int PWM = 3;


void setup() {

for(int pins = pinA; pins <= pinB; pins++) //This loop sets pins 4-10 as outputs.

{

pinMode(pins, OUTPUT);

}

pinMode(A0, INPUT);

pinMode(PWM, OUTPUT);

}


void loop() {

for(int pins = pinA; pins <= pinB; pins++) //Gives running lights

{

digitalWrite(pins, HIGH); //Turns the relevant light on and off in accordance with the ''pins'' variable.

delay(analogRead(A0));

digitalWrite(pins, LOW);

}

}
